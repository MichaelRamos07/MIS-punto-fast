# Proyecto PuntoFast: Sistema de Inteligencia de Negocios (MIS)

## Descripción del Proyecto
Este repositorio contiene la solución técnica desarrollada para **PuntoFast**, un sistema de Inteligencia de Negocios (MIS) diseñado para transformar datos transaccionales brutos en activos estratégicos. El proyecto automatiza el análisis de indicadores clave de desempeño (KPIs), permitiendo a la dirección del negocio tomar decisiones basadas en datos reales y precisos.

## Objetivos del Proyecto
* **Integración de Datos:** Centralizar la información proveniente de múltiples fuentes (ventas, inventarios, personal).
* **Análisis Estratégico:** Implementar KPIs clave para medir el ticket promedio, rotación de productos y desempeño del equipo de ventas.
* **Optimización Operativa:** Identificar áreas críticas, como horas pico y productos de baja rotación, para maximizar la rentabilidad.

## Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Análisis de Datos:** `pandas` (Procesamiento ETL y limpieza de datos).
* **Visualización:** `matplotlib
